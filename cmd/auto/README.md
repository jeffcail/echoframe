## 中文介绍

### 特点

- **快速生成**：支持根据配置文件快速生成代码，减少重复劳动。
- **高度可定制**：用户可以根据项目需求自定义代码模板。
- **支持多种语言**：支持语言Golang。

### 定制
```shell
cobra-cli add demo
```

#### 帮助
```shell
make help
```

#### 一键生成
```shell
make code
```

#### 控制器
```shell
make h
```

#### 服务层
```shell
make s
```

#### 数据层
```shell
make d
```
